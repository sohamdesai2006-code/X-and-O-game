# Tic Tac Toe Game 🎮

A beautifully designed, fully responsive Tic Tac Toe game built with HTML, CSS, and Vanilla JavaScript. 

### 🌐 Live Demo
**Play the game here:** [https://rock-paper-scissors-gamma-lovat.vercel.app/](https://rock-paper-scissors-gamma-lovat.vercel.app/)

## ✨ Features

- **Two Game Modes**:
  - **2 Player (vs Friend)**: Play locally with a friend.
  - **vs Computer**: Test your skills against an AI opponent.
- **Smart AI Opponent**:
  - **Normal Mode**: The computer makes random moves. Great for a casual game!
  - **Hard Mode**: The computer uses a strategic algorithm to block your winning moves and actively secure its own victories.
- **Customizable Gameplay**: Choose whether you want to play as **Player X** or **Player O**.
- **Modern UI/UX**:
  - Fully responsive design that looks great on desktop and mobile.
  - Smooth animations, interactive hover effects, and crisp drop shadows.
  - Disabled global text-selection to provide a native, app-like feel.
  - Dynamic full-screen Victory and Draw overlays.
- **Ready for Production**: easily deployable to Vercel or any static web host.

## 🚀 How to Play

1. Open `index.html` in your browser (or visit the live Vercel link).
2. Choose your preferred game mode from the home screen: **2 Player** or **vs Computer**.
3. Select your difficulty level and whether you want to play as 'X' or 'O'.
4. Take turns clicking on the 3x3 grid to place your mark.
5. Get 3-in-a-row (horizontally, vertically, or diagonally) to win!
6. If all 9 squares are filled and no one has 3-in-a-row, the game ends in a draw.

## 🛠️ Technologies Used

- **HTML5**: For the semantic structure of the game board and menus.
- **CSS3**: For the custom styling, text gradients, glassmorphism UI elements, and responsive layout.
- **JavaScript (ES6)**: For the core game state, DOM manipulation, turn management, and AI logic.
- **FontAwesome**: For sleek, professional UI icons.

## 📂 Project Structure

- `index.html`: Contains the layout for the menus and game board.
- `style.css`: Contains all styling, animations, and responsive rules.
- `x&o_app.js`: Contains the JavaScript logic running the game loop and AI.
